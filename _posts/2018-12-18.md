##  时间单位 (基于cpu指令)

** 我们假定CPU会顺序的执行所有的指令，而内存随机访问的代价是相同的 **

```javascript
const a = [1, 2, 3, 4, 5];

// 它的索引操作，就占用1个单位时间（也就是消耗1个CPU指令）
a[2]

const b = 1; // 消耗1个单位时间

const obj = {};
obj['x'] = 1; // 消耗1个单位时间

const d = new Date() // 消耗1个单位时间

```
