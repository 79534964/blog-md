## 时间单位 (基于cpu指令)

** 我们假定CPU会顺序的执行所有的指令，而内存随机访问的代价是相同的 **

```javascript
const a = [1, 2, 3, 4, 5];

// 它的索引操作，就占用1个单位时间（也就是消耗1个CPU指令）

a[2] // O(1)
a.slice(1) // O(n)
a.splice(1, 0, 7) // O(n)
a.concat() // O(n)
a.push // O(1)

```
