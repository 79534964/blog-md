## 介绍
** 1.  生成树形结构，表示“整体-部分”关系 **
** 2.  让整体和部分都具有一致的操作方式 **

## 代码演示
```javascript
class Adaptee {
	specificRequest() {
		return '德国标准的插头';
	}
};

class Target {
	constructor() {
		this.adaptee = new Adaptee();
	}
	request() {
		let info = this.adaptee.specificRequest();
		return `${info} -> 转换器 -> 中医标准的插头`;
	}
};

let target = new Target();
target.request();
```

## 应用场景
** 虚拟DOM中的Vnode是这种形式，但数据结构简单 **
